# 🚇 Metro Train Delay Prediction

## 📌 Overview
This project aims to predict metro train delays using machine learning based on schedule and operational data.

## 🎯 Objective
To build a model that predicts delay using features like:
- Time of day
- Station
- Peak hours
- Historical patterns

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib
